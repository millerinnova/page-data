List Vehicle/Unassigned Trips

Action​

List trips with vehicle Id or none (Unassigned)

API / State Management​

Data Flow: Describe how data is fetched, displayed, and managed on the page.

API Call​

End point: /location/trips

Request:​

  • User Identity (Authorization Key)

  • Query String Parameters

    • VehicleId: vehicle id (or unassigned if not given vehicleId)

    • pageIndex: 1

    • pageSize: 10

    • orderType: 0

    • orderBy: 0
